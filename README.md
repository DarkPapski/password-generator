# 🔐 Password Generator

უსაფრთხო პაროლების გენერატორი მოსახერხებელი ინტერფეისით.

## 📋 აღწერა

ეს არის მარტივი და ინტუიციური ვებ აპლიკაცია რანდომული პაროლების გენერირებისთვის. აპლიკაცია საშუალებას გაძლევთ შექმნათ უსაფრთხო პაროლები სხვადასხვა პარამეტრებით.

## ✨ ფუნქციები

- ⚙️ **მორგებადი სიგრძე** - აირჩიეთ პაროლის სიგრძე 4-დან 50 სიმბოლომდე
- 🔤 **დიდი ასოები** - ჩართეთ/გამორთეთ A-Z
- 🔢 **ციფრები** - ჩართეთ/გამორთეთ 0-9
- 🔣 **სპეციალური სიმბოლოები** - ჩართეთ/გამორთეთ !@#$%^&*()...
- 📊 **სამი ვარიანტი** - იხილეთ თქვენი კონფიგურაციით, ძლიერი და მარტივი პაროლები
- 🎨 **თანამედროვე დიზაინი** - მომხმარებელთან მეგობრული ინტერფეისი

## 🚀 გამოყენება

1. გახსენით `index.html` ფაილი ბრაუზერში
2. მიუთითეთ სასურველი პარამეტრები:
   - პაროლის სიგრძე
   - რომელი სიმბოლოები გსურთ
3. დააჭირეთ "პაროლის გენერირება" ღილაკს
4. დააკოპირეთ შემოთავაზებული პაროლი

## 🛠️ ტექნოლოგიები

- HTML5
- CSS3 (Gradient Design)
- Vanilla JavaScript

## 📦 ინსტალაცია

```bash
# კლონირება
git clone https://github.com/DarkPapski/password-generator.git

# გახსენით პროექტის ფოლდერი
cd password-generator

# გახსენით index.html ბრაუზერში
```
